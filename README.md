# B-13 A8 Summer Sale Project

##  Project Name
B-13 A8 Summer Sale

##  Purpose
This project is a simple eCommerce or product showcase web application built for a summer sale campaign.  
It allows users to browse products and view details in a clean and responsive UI. The goal is to practice modern full-stack web development and deploy it using Vercel.
